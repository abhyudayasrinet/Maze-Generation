
Maze Generator
--------------

Creates a square maze for a given input size N.


CODE.cpp
--------

Source code in C++

commands.txt
------------
Contains commands for GnuPlot required to create the maze of size N.
Sample output file generated for creating a maze of size 50.

nodes.txt
---------
Contains information of every node on the graph.
Sample output file generated for creating a maze of size 50.

process.txt
-----------
Information to show how DFS occurs in the code and the current node at every stage of the DFS
Sample output file generated for creating a maze of size 50.
